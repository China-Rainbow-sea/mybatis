# 多对一
多的的一方是:Student
一的一方是: Clazz

怎么分主表和副表
原则：谁在前看，谁就是主表

> 多对一：多在前，那么多就是主表，
> 一对多：一在前，那么一就是主表
> 

这里的一对多：关系
public class Student{}

public class Clazz { // 一个班级对象
// 一个班级对应多个学生
// 怎么去表示这个班级对应了多个学生对象呢
// 集合或者数组都可以容纳多个元素
List<Student> studentList
一对多，一在前，一是主表，多是副表
主表: t_clazz
副表: t_stu
}


多对多：分解成两个一对多，然后其中一个是：一对一的关系
